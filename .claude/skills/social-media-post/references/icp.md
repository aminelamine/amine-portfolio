# ICP Reference — Amine L.

**Profil cible** : Product Designer & AI Orchestration Designer, 40 ans, Paris.
15+ ans d'expérience. Travaille à l'intersection Design / Produit / IA sur des systèmes complexes.

**Ce qu'il cherche** : Des frameworks opérationnels pour structurer la collaboration humain–agent dans les produits. Pas des features AI — de l'orchestration système.

**Ce qui le trigger** :
- Stakeholders qui disent "on veut un agent IA" sans structure
- Équipes qui traitent l'IA comme une feature, pas comme un acteur système
- Absence de standard partagé pour : rôle agent, logique de décision, escalade, guardrails

**Ce qu'il "sauvegarde et réutilise"** :
- Templates de définition de rôle agent
- Canvas de flow agentic (états : confiant / partiel / incertain / refus / escalade)
- Frameworks Discovery → Delivery IA

**Comment il consomme LinkedIn** :
- Entre workshops : cherche des frameworks testables immédiatement
- Pendant ses cycles de réflexion : lit des deep dives sur l'orchestration
- Partage en interne : diagrammes système, behavior maps, templates adaptables

**Bullseyes éditoriaux** :
- "Designing the role of an AI agent inside a product"
- "Human vs AI : mapping responsibility before designing UI"
- "POC is not a product : scaling agentic systems"
- "Conversation design is not enough — think orchestration"
